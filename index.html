Dice Decide

Inspired by the philosophy of the cult classic "The Dice Man" by Luke Rhine heart. The user is encouraged to input up to six options into the boxes, the app then rolls a virtual dice and returns the corresponding box. The user can then chose save the outcome in their user page.
Built in React.js. The app was composed with Docker and uses both a PostgreSQL and a Redis database. The app also has Client Session management with JSON Web Tokens and authentication middleware.
